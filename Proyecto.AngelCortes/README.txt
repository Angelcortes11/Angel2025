Proyecto: Mini sistema de usuarios con roles

Estudiante: Ángel Bruno Cortes Hamie

Descripción del tema elegido: Pagina para crear usuarios indicando sus diferentes roles a traves de un formulario

Como ejecutar el Proyecto: 

1) Abrir Xampp e iniciar el servidor de apache
2) Abrir un buscador como Chrome y escribir:  localhost (y luego el nombre de la carpeta del Proyecto,) en este caso localhost/Proyecto.AngelCortes/
3) Darle enter y ya podrás ejecutar el programa de mini sistema de usuarios con sus roles.

Funciones implementadas:

Como funcion propia cree y usé "Permanencia" para mostrar cuanto tiempo ha estado el usuario en la empresa

Como funcion nativa usé strtoupper que simplemente sirve para convertir un String a mayusculas

Arrays implementados:

Usé los 3 tipos de Arrays: Asociativo ( cada uno de sus componentes es identificado
por una Key (Llave, en español) declarada manualmente. ), indexado ( cada componente es identificado por un índice numérico empezando
por 0 y incrementando por 1 con cada componente subsecuente. ) y multidimencional ( Arrays que contienen
otros Arrays como componentes. )
